# HomeLab-Documentation
A reposiroty to document the technologies used in my current Raspberry Pi homelab. 

# Overview
Designed, deployed and maintain a self-hosted Linux-based infrastructure environment running on a Raspberry Pi 5 using Docker and Docker Compose.

The environment hosts multiple internal and externally accessible services, providing hands-on experience with Linux administration, networking, reverse proxying, DNS management, SSL certificates, containerisation and security tooling.
